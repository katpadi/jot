# Jot

A tiny, minimalist browser note that removes the cost of context switching.

## Features

- Opens instantly from the toolbar
- Autosaves every keystroke
- Copy all to clipboard in one click
- Light and dark mode
- Notes are saved locally via `localStorage` — no account, no sync, no server.

## Install

[Chrome Web Store →Jot](https://chromewebstore.google.com/detail/jot/gogmghfcgficidobmgjbpfdejaaekgid)

## Development

1. Clone the repo
2. Go to `chrome://extensions` → enable **Developer mode**
3. Click **Load unpacked** → select the `jot/` folder

## Privacy

Jot does not collect any user data — this policy exists only because the Chrome Web Store requires one. See [Privacy Policy](https://katpadi.github.io/jot/privacy.html).
